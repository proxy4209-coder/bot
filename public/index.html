This Telegram Bot Cannot Run on Netlify

Netlify is built for static websites and short-lived serverless functions. A Telegram bot using Pyrogram requires a long-running process that stays connected to Telegram, which Netlify does not support.

Recommended Hosting Platforms

  • Render - Deploy with the existing Dockerfile. Free tier available.
  • Railway - Docker-based deployment, easy setup.
  • Fly.io - Container hosting with generous free tier.

Quick Start with Render

1. Push this repo to GitHub
2. Go to render.com and create a Web Service
3. Connect your GitHub repo
4. Set environment to Docker
5. Add environment variables:
BOT_TOKEN, API_ID, API_HASH
6. Deploy!
